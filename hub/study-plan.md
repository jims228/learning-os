## TOEFL 2026年新形式（重要）

**2026年1月以降、試験が大幅変更。古い教材で対策しないこと。**

| セクション | 新タスク | 時間 |
|-----------|--------|------|
| Reading | Complete the Words / Read in Daily Life / Read an Academic Passage | 30分 |
| Listening | Short conversations / Announcements / Academic Talks（45〜120秒） | 29分 |
| Writing | **Build a Sentence** / **Write an Email** / **Academic Discussion** | 23分 |
| Speaking | **Listen and Repeat**（7問）/ **Take an Interview**（4問） | 8分 |

全セクション適応型（正答率で難易度が自動調整される）。

## Listening強化の具体的な方法（最重要）

日本人がListeningで聞き取れない原因は「音声変化を知らないこと」。

知っている単語でも聞き取れない音声変化の例：
- **リンキング**：「want it」→「wanit」
- **リダクション**：「going to」→「gonna」
- **フラッピング**：「water」→「wader」（アメリカ英語）
- **脱落**：「next day」→「nekday」

**攻略手順：**

1. **英語耳**で発音の基礎（子音・母音）を体で覚える
2. **スタサプ**の音声変化ディクテーションで6種類の変化を認識訓練
3. 「知識」を「耳」に落とし込む：精聴→スクリプト確認→シャドーイング10回

**シャドーイングの正しいやり方（ATSUメソッドより）：**
1. スクリプトなしで音声を聞く
2. スクリプトを見て音声変化の箇所を確認
3. スクリプト見ながらシャドーイング（5〜10回）
4. スクリプトなしでシャドーイング（5〜10回）

## ノートテイキング（新形式でも必須）

TOEFLリスニングでメモをとる記号：

| 記号 | 意味 |
|------|------|
| → | causes / leads to |
| ↑↓ | increase / decrease |
| eg | example |
| def | definition |
| * | important |
| ? | unclear |

**メモは必ず英語でとる**（日本語変換の時間ロスを防ぐ）。

## Writing 新形式テンプレート（得意分野を確実に取る）

### Academic Discussion（最重要タスク）

**必須構成：**
1. 片方の学生に賛同（名前を引用）：`I agree with [Name]'s point that...`
2. 自分の意見＋根拠：1つの立場を一貫して主張
3. もう片方への言及：`[Name] makes a fair point, however...`

**目標語数**：120〜130語（長すぎても短すぎてもマイナス）

### Write an Email

**構成**：件名 → `Dear [Name],` → 目的 → 3つの指示事項を各1〜2文で対応 → 締め

## よくある失敗パターン（必読）

1. **単語だけやってListeningを放置** → 語彙と音声知覚は別スキル。Listenig練習は毎日必須
2. **BGM的に英語を流すだけ** → 精聴（スクリプト確認・シャドーイング）なしでは改善しない
3. **旧形式の教材で対策する** → 2026年以降の新タスク（Email・Build a Sentence等）に対応できない
4. **模試を受けない** → スコアを測定しないと強化すべき箇所が見えない。月2回は必須
5. **週末だけ大量にやる** → 毎日1.5時間＞週末10時間。特にListeningは連続入力が重要
6. **Writingの新形式を対策しない** → 得意なはずのWritingで新タスク対応が遅れると大損
