# learning-os

新後亮人の学習管理リポジトリ。
将来のBCI/宇宙/メタバーススタートアップと海外大学院進学に向けた基礎を積む。

## 目的

- 英語: 留学・海外PhD出願に向けてTOEFL100+を目指す
- BCI/宇宙/メタバース: 将来の研究・起業に必要な基礎知識の習得
- 数学: 機械学習・信号処理・神経科学に必要な基礎

## 毎週確認する指標

- 英語に毎日触れたか
- BCI/研究系の論文や知識を1つでも増やしたか
